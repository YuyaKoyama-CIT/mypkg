#トピックについて
* トピックとはノード間でやり取りするデータのパイプのようなものです。 相手の状態に関係なく常にデータを発信しており、不特定多数がデータを受け取ることができるので1対1通信だけでなく1対多数通信を行うことができます。 トピックに流れるデータのことをメッセージと言い、送信にはパブリッシャ、受信や読み取りにはサブスクライバを使用します。
